# omiseGohanProject
「おみせごはんプロジェクト」とは
福島県会津若松市と福島大学吉田ゼミナールによるタクシーフードデリバリーサービスである。

このwebアプリケーションは利用状況の分析の効率化のために開発している。
12/2/2022現在実装した機能：
1 配達地域と配達元飲食店の直線距離
2 googlemapでの車での運転距離
3　運転距離と福島県のタクシー料金規制から、その距離を移動するためのタクシー料金を試算。
